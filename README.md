
# IPL 2022 Data Analysis

This project analyzes IPL 2022 match-level data to evaluate team performance, match-winning factors, and standout players across the season. Using structured match and ball-by-ball statistics, the analysis examines how toss decisions, batting order, venue patterns, and individual performance metrics influenced match outcomes.
